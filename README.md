# نظام مسار الذكي ERP

واجهة عربية متجاوبة لنظام إدارة الموظفين، مبنية انطلاقًا من عرض «نظام مسار الذكي لإدارة الموظفين».

## التشغيل

لا يحتاج المشروع إلى تثبيت أي مكتبات أو تنفيذ عملية بناء.

1. افتح ملف `index.html` مباشرة في متصفح حديث.
2. أو شغّل خادمًا محليًا من داخل المجلد:

```bash
python3 -m http.server 8080
```

ثم افتح `http://localhost:8080`.

## الأقسام المتاحة

- لوحة تحكم تفاعلية ومؤشرات ورسوم تحليلية.
- إدارة الموظفين وملف الموظف الشامل 360°.
- البحث والتصفية وإضافة موظف وتصدير CSV.
- الحضور والانصراف والتسجيل اللحظي.
- الإجازات والاعتماد والرفض.
- الرواتب والمسيرات المالية.
- التقارير والتحليلات.
- الأمان والصلاحيات والإعدادات.
- الوضع الفاتح والداكن وتجربة جوال متجاوبة.

## ملاحظة تقنية

هذه نسخة واجهة أمامية جاهزة للعرض والتجربة، وتستخدم بيانات تجريبية داخل المتصفح. التشغيل الفعلي للرواتب، حسابات المستخدمين، قاعدة البيانات، البصمة، الخرائط والإشعارات يتطلب ربطها بخادم خلفي وقاعدة بيانات وواجهات API آمنة.
